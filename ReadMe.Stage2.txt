#category: Tutorial
My first Marmalade C++ App

The objectives of this tutorial series include:
    Build a Match-3 game (similar to games such as Bejeweled and Candy Crush) from scratch using the Marmalade SDK in stages
    Introduce key concepts of the Marmalade SDK as we progress from stage to stage
    Cover more advanced topics such as supporting in-app purchase, ads and social integration

Stage 2: Input and Audio

The aims of this tutorial stage include:
    Show how to monitor touch input and respond to screen touches
    Show how to play background music
    Show how to load and play sound effects
